# Real-Time Chat Application
A full-stack real-time chat application that enables instant messaging between users using WebSockets. Built with the MERN stack and Socket.IO, featuring secure authentication and dynamic UI updates.

## Features
- Real-time messaging using Socket.IO  
- User authentication with JWT  
- Secure password hashing (bcrypt)  
- Dynamic chat interface (React)  
- Persistent chat messages stored in MongoDB  
- User-specific conversations

## Tech Stack
Frontend: React.js, Tailwind CSS  
Backend: Node.js, Express.js  
Database: MongoDB  
Real-Time: Socket.IO  
Authentication: JWT, bcrypt  

## Live Demo
https://real-time-chat-five-fawn.vercel.app/

## How It Works
- Users authenticate using JWT-based login  
- Socket.IO establishes real-time connection between clients and server  
- Messages are emitted and received instantly without page reload  
- Chat data is stored in MongoDB for persistence

## Future Improvements
- Group chat functionality  
- Typing indicators  
- Message read receipts  
- Media/file sharing
